AI Concept Map

This map is an attempt to form a comprehensive overview of the current machine learning space. There are many related concepts in machine learning, and this visualization aims to inform beginners and experts alike on these connections. It starts from the "most general" fields of machine learning and dives as far down as specific algorithms to accomplish a particular goal.

    How to Use

    2. Double-click a node to expand or collapse its relations
    3. Click and drag a node around to pin it
    4. Middle click to unpin a node
